# Hi, I'm Brian Castle
Senior/staff-level software engineer focused on backend/platform architecture, Ruby on Rails, APIs, billing and identity
systems, code quality, and developer productivity.

## What I work on
- Enterprise SaaS applications and backend platforms
- Ruby/Rails services, APIs, admin tools, background jobs, and data workflows
- Billing, payments, identity, authentication, authorization, and internal tooling
- Code quality, tests, refactoring, review culture, and maintainable system design

## Core technologies
Ruby, Rails, PostgreSQL, MySQL, Sidekiq, Redis, REST, RSpec, JavaScript/TypeScript, React.js,
AWS, Datadog, Git/GitHub, CI/CD.

While those are my most recent focus, I have worked with many languages (Ruby, Java, Perl, Go, etc)
across many stacks/frameworks (Rails, JEE), and worked with many 3rd-party services from Auth0 to Stripe.

## How I like to work
I like ambiguous product/platform problems, clear design docs, incremental migrations, strong tests, practical code
reviews, technical communication, and teams that value shared ownership. Most of my professional work is in private
company repositories, so this profile is primarily a pointer to my background and interests.
